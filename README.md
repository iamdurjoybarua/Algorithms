# Algorithms Lab

This repository contains my implementations and analysis of fundamental algorithms developed during the **Algorithms Lab** sessions. Each implementation focuses on understanding algorithmic paradigms, optimizing time/space complexity, and following clean coding practices.

---

## Repository Structure

The implementations are organized by algorithmic technique:

* **`Searching-Sorting/`**: Linear Search, Binary Search, Merge Sort, Quick Sort, Heap Sort.
* **`Divide-and-Conquer/`**: Closest Pair of Points, Strassen’s Matrix Multiplication.
* **`Greedy-Method/`**: Huffman Coding, Fractional Knapsack, Prim's and Kruskal’s algorithms.
* **`Dynamic-Programming/`**: 0/1 Knapsack, Longest Common Subsequence (LCS), Matrix Chain Multiplication.
* **`Graph-Theory/`**: Breadth-First Search (BFS), Depth-First Search (DFS), Dijkstra’s, Bellman-Ford.
* **`Backtracking-BranchBound/`**: N-Queens Problem, Sum of Subsets, Traveling Salesperson (TSP).
